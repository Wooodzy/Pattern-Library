Grid Elements

Container

The Container is one of the main building blocks of a website and are used to contain grids or grid items such as coloumns and rows within.

<body>
<div class= "Container" >
</div>
</body>

Row

The row is next part of the grid system, this acts as a wrapper for the columns, there is also the grid-row property that is used to specify a grid items size and location.

<div class= "row" >
</div>

Column

There are two different ways you can create a grid using coloumns, using multiple divs or the <col>tag or Column-6 for example this lets you customise the width because using <div> will create equal size grid layout.

<div class= "row" >
<div class= "column" >
</div>
</div>
