Navigation Bar Structure

The navigation is normally placed in the <header> as common practice but there is no rule for this but generally depends on where you want the navigation to be on the site. Before HTML 5 there would be no <nav> tag.

<div id= 'navbar' >
<ul>
<li> <a href= '#' <span> Home </span> </a> </li>
<li> <a href= '#' <span> LINK 1 </span> </a> </li>
<li> <a href= '#' <span> LINK 2 </span> </a> </li>
<li> <a href= '#' <span> LINK 3 </span> </a> </li>
</ul>
</nav>
